# RESTFul API for Cupcakes ordering & selling company
## Node.js with Express.js
### Installation
- Clone
```bash
git clone https://github.com/nilupul-madhusanka/nms-cupcake-api.git
```
- Go to folder
```bash
cd nms-cupcake-api
```
- Init
```bash
npm init
```
- Install Dependencies
```bash
npm i express
```
- Run API
```bash
node app.js
```
## Endpoint Functions
- Get all cupcake items
- Add new cupcake item
- Edit new cupcake item
- Delete new cupcake item
- Get all customers
- Create new customer
- Create an order for new customer
- Get all orders for selected customer
- Filter all orders which has more than 10 quantity in an order. (10 can be a variable)
- Filter top 5 customers which have higher sales
- Find the top order (highest value order) of a selected customer
