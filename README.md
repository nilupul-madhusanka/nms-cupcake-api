# RESTFul API for Cupcakes ordering & selling company
## Node.js with Express.js
### Installation
- Clone
```bash
git clone https://github.com/nilupul-madhusanka/nms-cupcake-api.git
```
- Go to folder
```bash
cd nms-cupcake-api
```
- Init
```bash
npm init
```
- Install Dependencies
```bash
npm i express
```
- Run API
```bash
node app.js
```
